Select current menu.<br>
<hr><br>
Call variants:<br><br>
<b>bool menu.Select</b>(index);<br>
<b>bool menu.Select</b>('name');<br>
<hr><br>
Arguments:
<ul style="margin-left: -20px">
    <li><b>index</b> - integer - menu item number for index selection<br>
    <li><b>name</b> - string - the name of the item in the menu for selection by name<br>
</ul>
<hr><br>
<b>Result:</b> boolean - true if menu is selected.
