Hide object.<br>
<hr><br>
Call variants:<br><br>
<b>Orion.Hide</b>();<br>
<b>Orion.Hide</b>('serial');<br>
<hr><br>
Arguments:
<ul style="margin-left: -20px">
    <li><b>serial</b> - string - object serial number to hide<br>
        <i>Default value</i>: <b>call a target to specify the object manually</b><br>
</ul>
<hr><br>
<b>Result:</b> does not return a value.
