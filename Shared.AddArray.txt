Add a global array to shared memory.<br>
<hr><br>
<b>Shared.AddArray</b>('name', array);<br>
<hr><br>
Arguments:
<ul style="margin-left: -20px">
    <li><b>name</b> - string - global array name<br>
    <li><b>array</b> - array, the value of the global array<br>
</ul>
<hr><br>
<b>Result:</b> does not return a value.
