Add an active click zone to the custom gump.<br>
<hr><br>
Call variants:<br><br>
<b>customGump.AddHitBox</b>(serial, x, y, width, height);<br>
<b>customGump.AddHitBox</b>(serial, x, y, width, height, callOnMouseUp);<br>
<b>customGump.AddHitBox</b>(serial, x, y, width, height, callOnMouseUp, toPage);<br>
<hr><br>
Arguments:
<ul style="margin-left: -20px">
    <li><b>serial</b> - integer - item serial number. Returned in response to the gump as the ReturnCode<br>
    <li><b>x</b> - integer - X coordinate<br>
    <li><b>y</b> - integer - Y coordinate<br>
    <li><b>width</b> - integer - width<br>
    <li><b>height</b> - integer - height<br>
    <li><b>callOnMouseUp</b> - integer - mouse release call. If 0 is specified - triggered when the mouse is pressed, if 1 - when released<br>
        <i>Default value</i>: <b>0</b><br>
    <li><b>toPage</b> - integer - page number to go. If not -1 is specified, it works like a page switch<br>
        <i>Default value</i>: <b>-1</b><br>
</ul>
<hr><br>
<b>Result:</b> does not return a value.
