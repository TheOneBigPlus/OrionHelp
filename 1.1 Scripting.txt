The <i>include</i> system:
<br><br>
At the top of a script file, write <b>//#include path/to/file</b><br>
If the syntax is correct, the line will be highlighted in <label style="color:red;">red</label>.
<br><br>
Includes are taken into account when loading/saving/reloading the current script file and when starting a new script if changes have been made to the code (the code itself is not saved, it is recommended to manually save the files).
<br><br>
There are two possible search systems for the include module:<br>
1 - Search <i>relative</i> to the current file being processed (includes may also contain links to other includes. The search path in the first step is calculated relative to the current module being processed).<br>
2 - Search by the <i>direct path</i> specified in the line (if a relative path is specified, instead of the direct path, the file can be found if it lies in the folder where the application was launched from).
<br><br>
The same module (along the same path) is connected only 1 time.<br>
If the file is not found, a corresponding message will be displayed in the game’s system chat (when saving the script).
<br><br>
An example of including a module in a relative path (from a folder with the current script):<br>
<b><label style="color:red;">//#include common.oajs</label></b>
<br><br>
An example of including a module in a relative path (from folder includes):<br>
<b><label style="color:red;">//#include includes/common.oajs</label></b>
<br><br>
An example of including a module in a relative path (the directory above where the folder is located includes):<br>
<b><label style="color:red;">//#include ../includes/common.oajs</label></b>
<br><br>
An example of including a module in a direct way:<br>
<b><label style="color:red;">//#include E:/OrionScripts/includes/common.oajs</label></b>
<br>
<hr>
Implemented support for secure includes.<br>
In the context menu of the script editor, you can select the <b>Make protected include file</b> item, and select the file you want to protect.<br>
A file will be created with the extension <b>*.jsp</b> or <b>*.oajsp</b><br>
This file can be used as a include:<br>
<b><label style="color:red;">//#include common.oajsp</label></b>
