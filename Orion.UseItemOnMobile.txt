Use an item with the new targeting system.<br>
The new targeting system must be supported by the server to work correctly.<br>
<hr><br>
<b>Orion.UseItemOnMobile</b>('serial', 'targetSerial');<br>
<hr><br>
Arguments:
<ul style="margin-left: -20px">
    <li><b>serial</b> - string - serial number of the item to use<br>
    <li><b>targetSerial</b> - string - serial number of the mobile<br>
</ul>
<hr><br>
<b>Result:</b> does not return a value.
