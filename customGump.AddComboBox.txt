Add dropdown list to custom gump.<br>
<hr><br>
Call variants:<br><br>
<b>customGump.AddComboBox</b>(serial, x, y, 'graphic', compositeBackground, 'openGraphic', width, textOffsetY);<br>
<b>customGump.AddComboBox</b>(serial, x, y, 'graphic', compositeBackground, 'openGraphic', width, textOffsetY, showItemsCount);<br>
<b>customGump.AddComboBox</b>(serial, x, y, 'graphic', compositeBackground, 'openGraphic', width, textOffsetY, showItemsCount, centerMaximized);<br>
<hr><br>
Arguments:
<ul style="margin-left: -20px">
    <li><b>serial</b> - integer - item serial number<br>
    <li><b>x</b> - integer - X coordinate<br>
    <li><b>y</b> - integer - Y coordinate<br>
    <li><b>graphic</b> - string - the background number of the dropdown when closed<br>
    <li><b>compositeBackground</b> - integer - 0 - resizepic is used, 1 - a combined background of 3 images is used<br>
    <li><b>openGraphic</b> - string - background number of the drop-down list in the open state<br>
    <li><b>width</b> - integer - dropdown width<br>
    <li><b>textOffsetY</b> - integer - dropdown text offset<br>
    <li><b>showItemsCount</b> - integer - number of displayed items open<br>
        <i>Default value</i>: <b>5</b><br>
    <li><b>centerMaximized</b> - integer - center maximizer<br>
        <i>Default value</i>: <b>0</b><br>
</ul>
<hr><br>
<b>Result:</b> does not return a value.
