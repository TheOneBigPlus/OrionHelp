Add a gump picture to a custom gump.<br>
<hr><br>
Call variants:<br><br>
<b>customGump.AddGumpPic</b>(x, y);<br>
<b>customGump.AddGumpPic</b>(x, y, 'graphic', 'color');<br>
<b>customGump.AddGumpPic</b>(x, y, 'graphic', 'color', serial);<br>
<b>customGump.AddGumpPic</b>(x, y, 'graphic', 'color', serial, 'colorSelected');<br>
<b>customGump.AddGumpPic</b>(x, y, 'graphic', 'color', serial, 'colorSelected', width, height);<br>
<hr><br>
Arguments:
<ul style="margin-left: -20px">
    <li><b>x</b> - integer - X coordinate<br>
    <li><b>y</b> - integer - Y coordinate<br>
    <li><b>graphic</b> - string - picture number<br>
    <li><b>color</b> - string - color<br>
        <i>Default value</i>: <b>'0'</b><br>
    <li><b>serial</b> - integer - item serial number. If a value other than 0 is specified, the picture works like a button<br>
        <i>Default value</i>: <b>0</b><br>
    <li><b>colorSelected</b> - string - color in the selected state, works only if serial is not zero<br>
        <i>Default value</i>: <b>'0'</b><br>
    <li><b>width</b> - integer - width to calculate the number of repetitions. The picture itself is not scaled. If 0 is specified - the width of the original image is taken<br>
        <i>Default value</i>: <b>0</b><br>
    <li><b>height</b> - integer - height to calculate the number of repetitions. The picture itself is not scaled. If 0 is specified, the height of the original image is taken<br>
        <i>Default value</i>: <b>0</b><br>
</ul>
<hr><br>
<b>Result:</b> does not return a value.
