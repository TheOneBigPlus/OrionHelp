Link timer to object.<br>
<hr><br>
<b>Orion.DisplayTimerSetObject</b>('id', 'serial');<br>
<hr><br>
Arguments:
<ul style="margin-left: -20px">
    <li><b>id</b> - string - display timer ID<br>
    <li><b>serial</b> - string - serial number of object to link<br>
</ul>
<hr><br>
<b>Result:</b> does not return a value.
