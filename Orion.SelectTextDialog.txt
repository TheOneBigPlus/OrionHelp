Make a selection in a text dialog.<br>
<hr><br>
Call variants:<br><br>
<b>bool Orion.SelectTextDialog</b>('name', 'answer');<br>
<b>bool Orion.SelectTextDialog</b>('serial', 'id', 'name', 'answer');<br>
<hr><br>
Arguments:
<ul style="margin-left: -20px">
    <li><b>name</b> - string - the name of the dialog<br>
        When using <b>'any'</b> or an empty string - this parameter is ignored<br>
    <li><b>serial</b> - string - serial number of the dialog<br>
        When using <b>'any'</b> - this parameter is ignored<br>
    <li><b>id</b> - string - dialog id<br>
        When using <b>'any'</b> - this parameter is ignored<br>
    <li><b>answer</b> - line, response text<br>
        Use <b>'cancel'</b> or an empty string to close the dialog with the cancel button, if available<br>
</ul>
<hr><br>
<b>Result:</b> boolean - true if the choice is made.
