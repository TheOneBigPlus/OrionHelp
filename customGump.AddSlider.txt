Add slider to custom gump.<br>
<hr><br>
<b>customGump.AddSlider</b>(serial, x, y, 'graphic', 'graphicSelected', 'graphicPressed', 'backgroundGraphic', compositeBackground, vertical, width, minValue, maxValue, value);<br>
<hr><br>
Arguments:
<ul style="margin-left: -20px">
    <li><b>serial</b> - integer - item serial number<br>
    <li><b>x</b> - integer - X coordinate<br>
    <li><b>y</b> - integer - Y coordinate<br>
    <li><b>graphic</b> - string - image number in normal state<br>
    <li><b>graphicSelected</b> - string - number of the image in the selected state (mouse over the element)<br>
    <li><b>graphicPressed</b> - string - image number in the pressed state<br>
    <li><b>backgroundGraphic</b> - string - background graphic<br>
    <li><b>compositeBackground</b> - integer - 0 - resizepic is used, 1 - a combined background of 3 images is used<br>
    <li><b>vertical</b> - integer - 0 - horizontal, 1 - vertical<br>
    <li><b>width</b> - integer - width<br>
    <li><b>minValue</b> - integer - minimum value<br>
    <li><b>maxValue</b> - integer - maximum value<br>
    <li><b>value</b> - integer - starting value<br>
</ul>
<hr><br>
<b>Result:</b> does not return a value.
