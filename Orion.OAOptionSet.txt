Change the value of an Orion assistant option.<br>
<hr><br>
<b>Orion.OAOptionSet</b>('option', 'value');<br>
<hr><br>
Arguments:
<ul style="margin-left: -20px">
    <li><b><a href="search:AssistantOptions">option</a></b> - string - option name<br>
    <li><b>value</b> - string - value to set (for boolean data types - "0" or "1")<br>
</ul>
<hr><br>
<b>Result:</b> does not return a value.
