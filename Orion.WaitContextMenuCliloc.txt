Add a wait hook for a context menu.<br>
<hr><br>
<b>Orion.WaitContextMenuCliloc</b>('serial', clilocID);<br>
<hr><br>
Arguments:
<ul style="margin-left: -20px">
    <li><b>serial</b> - string - the serial number of the context menu to wait for.<br>
        '0' - ignore serial number check<br>
    <li><b>clilocID</b> - integer - the number of the cliloc whose text is displayed in the menu (can be found by the InfoContextMenu() function)<br>
</ul>
<hr><br>
<b>Result:</b> does not return a value.
