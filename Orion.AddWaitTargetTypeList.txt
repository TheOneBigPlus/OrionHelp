Add to the queue a target hook for an object found by a search in the container.<br>
<hr><br>
Call variants:<br><br>
<b>Orion.AddWaitTargetTypeList</b>('findListName');<br>
<b>Orion.AddWaitTargetTypeList</b>('findListName', 'container', 'flags');<br>
<b>Orion.AddWaitTargetTypeList</b>('findListName', 'container', 'flags', recurse);<br>
<hr><br>
Arguments:
<ul style="margin-left: -20px">
    <li><b>findListName</b> - string - search list name<br>
    <li><b>container</b> - string - the serial or alias of the container to search<br>
        <i>Default value</i>: <b>'self'</b><br>
    <li><b><a href="search:FindFlags">flags</a></b> - string - search flags<br>
        <i>Default value</i>: <b>''</b><br>
    <li><b>recurse</b> - boolean - true if you want to search pre-opened subcontainers<br>
        <i>Default value</i>: <b>true</b><br>
</ul>
<hr><br>
<b>Result:</b> does not return a value.
