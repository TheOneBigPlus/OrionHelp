Resume a script or all scripts.<br>
<hr><br>
<b>Orion.ResumeScript</b>('name');<br>
<b>Orion.ResumeScript</b>('name', 'exceptScripts');<br>
<hr><br>
Arguments:
<ul style="margin-left: -20px">
    <li><b>name</b> - string - script name to resume.<br>
        <b>'all'</b> - resume all paused scripts<br>
    <li><b>exceptScripts</b> - string - functions to keep paused.  Use this in combination with the 'all' flag.<br>
        The names of the scripts are a case-sensitive string, with pipe separation: | <br>
        Example: <i>Orion.ResumeScript('all', 'Heal|Loot|CheckMana');</i><br>
        <i>Default value</i>: <b>''</b><br>
</ul>
<hr><br>
<b>Result:</b> does not return a value.
