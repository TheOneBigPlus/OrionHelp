Add a button with a static image to a custom gump.<br>
<hr><br>
Call variants:<br><br>
<b>customGump.AddButtonTileArt</b>(serial, x, y, 'graphic', 'graphicSelected', 'graphicPressed', 'color', 'tileGraphic', 'tileColor', tileX, tileY, action);<br>
<b>customGump.AddButtonTileArt</b>(serial, x, y, 'graphic', 'graphicSelected', 'graphicPressed', 'color', 'tileGraphic', 'tileColor', tileX, tileY, action, toPage);<br>
<hr><br>
Arguments:
<ul style="margin-left: -20px">
    <li><b>serial</b> - integer - item serial number. Returned in response to the gump as ReturnCode<br>
    <li><b>x</b> - integer - X coordinate<br>
    <li><b>y</b> - integer - Y coordinate<br>
    <li><b>graphic</b> - string - image number in normal state<br>
    <li><b>graphicSelected</b> - string - number of the image in the selected state (mouse over the element)<br>
    <li><b>graphicPressed</b> - string - image number in the pressed state<br>
    <li><b>color</b> - string - button color<br>
    <li><b>tileGraphic</b> - string - static image number<br>
    <li><b>tileColor</b> - string - static color<br>
    <li><b>tileX</b> - integer - X coordinate for static<br>
    <li><b>tileY</b> - integer - Y coordinate for static<br>
    <li><b>action</b> - integer - action type, 0 - go to the page, 1 - selection in the gump<br>
        <i>Default value</i>: <b>1</b><br>
    <li><b>toPage</b> - integer - page number for action=0<br>
        <i>Default value</i>: <b>-1</b><br>
</ul>
<hr><br>
<b>Result:</b> does not return a value.
