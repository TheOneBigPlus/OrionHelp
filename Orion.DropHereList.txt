Find an object from a search list and drop the item under your feet.<br>
<hr><br>
Call variants:<br><br>
<b>bool Orion.DropHereList</b>('listName');<br>
<b>bool Orion.DropHereList</b>('listName', 'container');<br>
<b>bool Orion.DropHereList</b>('listName', 'container', count);<br>
<hr><br>
Arguments:
<ul style="margin-left: -20px">
    <li><b>listName</b> - string - search list name. You can add new lists in the Lists tab<br>
    <li><b>container</b> - string - the serial or alias of the container to search<br>
        <i>Default value</i>: <b>'backpack'</b><br>
    <li><b>count</b> - integer - how many items to drop<br>
        0 - drop all<br>
        <i>Default value</i>: <b>0</b><br>
</ul>
<hr><br>
<b>Result:</b> boolean - true if the item is found.
