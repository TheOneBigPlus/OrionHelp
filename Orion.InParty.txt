Check if the object is in a party<br>
<hr><br>
Call variants:<br><br>
<b>bool Orion.InParty</b>();<br>
<b>bool Orion.InParty</b>('serial');<br>
<hr><br>
Arguments:
<ul style="margin-left: -20px">
    <li><b>serial</b> - string - serial number of the object to be checked<br>
    <i>Default value</i>: <b>'self'</b><br>
</ul>
<hr><br>
<b>Result:</b> boolean - true if the object (character) is in a party
