Add text to custom gump.<br>
Text supports <a href="search: HTMLTags">HTML markup</a>, used in the UO.<br>
Text supports auto-update attributes:
<ul style="margin-left: -20px">
    <li><b>{name}</b> - displays the name of the character<br>
    <li><b>{namenoto}</b> - displays the name of the character and paints it in the color of notoriety<br>
    <li><b>{hits}</b> - outputs the current value Hitpoints<br>
    <li><b>{maxhits}</b> - outputs the current value MaxHitpoints<br>
    <li><b>{hits%}</b> - outputs the current value Hitpoints в процентах<br>
    <li><b>{mana}</b> - outputs the current value Mana<br>
    <li><b>{maxmana}</b> - outputs the current value MaxMana<br>
    <li><b>{mana%}</b> - outputs the current value Mana в процентах<br>
    <li><b>{stam}</b> - outputs the current value Stamina<br>
    <li><b>{maxstam}</b> - outputs the current value MaxStamina<br>
    <li><b>{stam%}</b> - outputs the current value Stamina в процентах<br>
    <li><b>{weight}</b> - outputs the current value Weight<br>
    <li><b>{maxweight}</b> - outputs the current value MaxWeight<br>
    <li><b>{weight%}</b> - outputs the current value Weight в процентах<br>
    <li><b>{str}</b> - outputs the current value Str<br>
    <li><b>{dex}</b> - outputs the current value Dex<br>
    <li><b>{int}</b> - outputs the current value Int<br>
    <li><b>{armor}</b> - outputs the current value Armor<br>
    <li><b>{gold}</b> - outputs the current value Gold<br>
    <li><b>{followers}</b> - outputs the current value Followers<br>
    <li><b>{maxfollowers}</b> - outputs the current value MaxFollowers<br>
    <li><b>{item:graphic,color}</b> - displays the current number of items of the specified type and color in the character's backpack (including in open containers and on the character himself)<br>
        color can be omitted or specified as 0xFFFF. Then the search takes place in all colors.<br>
        It is allowed to indicate several types and colors of objects, separating them |<br>
        For example: "Total ores: {item:0x19B7|0x19B8|0x19B9|0x19BA}" or "Total iron and copper ores: {item:0x19B7|0x19B8|0x19B9|0x19BA,0|0x0026|0x0590}"<br>
</ul>
<hr><br>
Call variants:<br><br>
<b>customGump.AddText</b>(x, y, 'color', 'text');<br>
<b>customGump.AddText</b>(x, y, 'color', 'text', width);<br>
<b>customGump.AddText</b>(x, y, 'color', 'text', width, serial);<br>
<hr><br>
Arguments:
<ul style="margin-left: -20px">
    <li><b>x</b> - integer - X coordinate<br>
    <li><b>y</b> - integer - Y coordinate<br>
    <li><b>color</b> - string - text color<br>
    <li><b>text</b> - string - text<br>
    <li><b>width</b> - integer - width of the text, 0 - unlimited in width<br>
        <i>Default value</i>: <b>0</b><br>
    <li><b>serial</b> - integer - item serial number<br>
        <i>Default value</i>: <b>0</b><br>
</ul>
<hr><br>
<b>Result:</b> does not return a value.
